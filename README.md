## BugImage - Defect Showcase & Admin

A production-ready web app to showcase product defects with a beautiful public viewer (carousel + detail pages) and an admin module for CRUD, uploads, visibility toggles, and ordering.

Stack: Next.js (App Router) + TypeScript, Tailwind CSS, Prisma ORM, PostgreSQL, Zod, bcrypt, Framer Motion. Pluggable storage (local by default, cloud-ready).

See the bottom of this file for local run instructions.
